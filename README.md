Food Delivery Business Analytics Dashboard
Project Overview

This project focuses on analyzing food delivery business operations using Excel, SQL Server, and Power BI. It provides an interactive dashboard to explore revenue patterns, customer behavior, delivery performance, and payment preferences.

The goal is to transform raw business data into meaningful insights that support better decision-making.

Business Problem

Food delivery businesses face challenges such as:

improving delivery efficiency
understanding customer ordering behavior
tracking revenue performance
optimizing payment and operational trends

This dashboard helps in analyzing these areas through structured KPIs and visual insights.

Tech Stack
Microsoft Excel (Data preparation)
SQL Server (Data analysis & querying)
Power BI (Dashboard & visualization)
Key Features
Revenue and sales performance tracking
Order trend analysis over time
Delivery performance insights
Payment method distribution analysis
Customer segmentation by age group
Interactive filters, slicers, and KPI cards
SQL Analysis

SQL was used to perform structured data analysis, including:

revenue aggregation and calculations
customer segmentation based on behavior
order and delivery performance metrics
payment method analysis
business KPI extraction
Dashboard Preview

Key Business Insights
UPI emerged as the most preferred payment method
Medium delivery category handled the majority of orders
Customers aged 36–50 contributed the highest number of orders
Revenue trends showed variation across different restaurant categories
Outcome

This project helped in understanding how data analytics can improve operational efficiency and business decision-making in the food delivery domain.
