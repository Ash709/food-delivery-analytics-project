## 📌 Project Overview
This project focuses on analyzing food delivery business operations using Excel, SQL Server, and Power BI. It provides an interactive dashboard to explore revenue patterns, customer behavior, delivery performance, and payment preferences.

The goal is to transform raw business data into meaningful insights that support better decision-making.

---

## 🎯 Business Problem
Food delivery businesses face challenges such as:
- improving delivery efficiency  
- understanding customer ordering behavior  
- tracking revenue performance  
- optimizing payment and operational trends  

This dashboard helps in analyzing these areas through structured KPIs and visual insights.
